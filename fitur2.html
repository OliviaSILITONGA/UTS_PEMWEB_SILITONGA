Langkah-langkah dasar membuat website sendiri.

Langkah 1: Buat Struktur HTML

Buat file index.html dan susun bagian utama seperti <header>, <main>, dan <footer>.

Langkah 2: Tambahkan Gaya dengan CSS

Buat file style.css untuk mempercantik tampilan website menggunakan warna, ukuran font, dan tata letak.

Langkah 3: Tambahkan Interaktivitas dengan JavaScript

Gunakan JavaScript untuk menambahkan animasi, tombol yang berfungsi, atau mode gelap seperti di website ini.

Langkah 4: Uji dan Publikasikan

Setelah selesai, uji tampilan di browser, lalu upload ke GitHub Pages atau Vercel agar bisa diakses online.
