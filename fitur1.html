Dasar utama dalam membangun tampilan dan interaktivitas web.

HTML (HyperText Markup Language)

HTML adalah bahasa yang digunakan untuk menampilkan dokumen web. HTML bukanlah bahasa pemrograman, melainkan bahasa markah, dikarenakan HTML tidak memiliki variabel fungsi, dan sebagian besar variabel-variabelnya merupakan bagian dari bahasa pemrograman. Setiap elemen dibentuk dengan tag seperti <h1>, <p>, dan <img>.

HTML berguna untuk menampilkan konten, menghubungkan (link) antar halaman, memberi struktur dan informasi terkait dengan sebuah halaman web. Konten sebuah web tidak hanya terbatas pada teks saja, melainkan konten interaktif lainnya seperti video, audio, gambar dan animasi dapat disisipkan dan ditampilkan pada halaman web.

CSS (Cascading Style Sheets)

CSS ini adalah bahasa dalam pemrograman untuk menentukan bagaimana dokumen itu disajikan. CSS berfungsi untuk menjelaskan dan menata tampilan elemen yang tertulis pada bahasa markup, salah satunya adalah HTML. Dalam arti lain, CSS-lah yang menjelaskan bagaimana elemen HTML di balik layar ditampilkan sedemikian rupa di layar website nanti. Baik HTML maupun CSS, keduanya saling melengkapi.

CSS berfungsi: memisahkan konten dan desain, pengaturan tampilan visual, desain responsif, meningkatkan aksesbilitas, mengurangi ukuran file HTML, desain konsisten, mendukung reusability, serta meningkatkan User Experience (UX).

JavaScript

JavaScript termasuk bahasa serbaguna yang bisa dipakai full-stack, dari tampilan hingga pengolahan data di backend. Javascript merupakan bahasa script yang dimana bahasa script adalah bahasa pemrograman yang ringan.

JavaScript berfungsi untuk memberi ineraktivitas pada website, mampu memanipulasi DOM, serta bisa berkomunikasi dengan server tanpa harus me-reload halaman.

Tailwind CSS

Tailwind adalah sebuah utility-first CSS framework yang memungkinkan pengembang untuk membangun desain yang kompleks dengan mudah. Tailwind memberikan utility classes yang dapat digunakan untuk menyesuaikan tampilan elemen HTML dengan cepat.

Tailwind dipilih karena efisiensi dan fleksibilitas. Tailwind juga membantu menjaga ukuran file akhir tetap kecil. Tailwind juga memudahkan kolaborasi dengan tim.
